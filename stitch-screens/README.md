## Fetching Assets

To download the latest assets from Stitch, run:

```bash
STITCH_API_KEY=your_key node fetch-stitch-assets.mjs
```

Get your API key at: https://stitch.withgoogle.com/ → Profile → Stitch Settings → API Keys
